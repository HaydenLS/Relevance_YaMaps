# Project Overview

Репозиторий содержит эксперименты по классификации пользовательских запросов относительно организаций на картах с использованием базовой модели и LLM-агентов.
Решение построено в три этапа: baseline-модель, LLM-агент и многошаговый агент с веб-поиском.

# Структура проекта
Папка `/notebooks/` содержит:
1. `1_baseline.ipynb` - Базовая трансформер модель для классификации запросов. Полученные результаты в конце.
2. `2_llm_agent.ipynb` - Решение задачи классификации с использованием простого запроса в LLM.
3. `3_agent.ipynb` - Базовая версия многошагового агента с маршрутизацией и возможностью веб-поиска.
Далеее этот агент был доработан и перенесен в `org_relevance`
