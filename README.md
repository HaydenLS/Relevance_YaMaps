## Project Overview

Репозиторий содержит эксперименты по классификации пользовательских запросов относительно организаций с использованием базовой модели и LLM-агентов.  
Решение построено в три этапа: baseline-модель, LLM-агент и многошаговый агент с веб-поиском.

---

## Структура проекта

### 1. Baseline (`1_baseline.ipynb`)
Базовая модель для классификации запросов.

Основные шаги:
- Подготовка и очистка текстовых данных.
- Анализ распределения длин последовательностей.
- Подбор параметров входных данных.
- Обучение и оценка модели.

Результат:
- Реализована и обучена базовая модель классификации.
- Получена стабильная конфигурация модели и входных данных.
- Сформирован базовый уровень качества для сравнения с последующими решениями.

---

### 2. LLM Agent (`2_llm_agent.ipynb`)
Решение задачи классификации с использованием LLM-агента.

Основные шаги:
- Построение пайплайна на основе LLM.
- Эксперименты с различными режимами работы модели.
- Настройка параметров запросов и батчей.
- Оценка качества решений.

Результат:
- Реализован LLM-агент для классификации запросов.
- Получено небольшое улучшение качества относительно baseline-подхода.
- Подтверждена применимость LLM для данной задачи.

---

### 3. Multi-step Agent with Web Search (`3_agent.ipynb`)
Многошаговый агент с маршрутизацией и возможностью веб-поиска.

Архитектура:
- Router-узел для решения о необходимости веб-поиска.
- Веб-поиск при недостатке локальной информации.
- Финальный классификатор на основе собранных данных.

Результат:
- Реализован агент с динамической маршрутизацией.
- Система корректно обрабатывает как простые, так и неоднозначные запросы.
- Веб-поиск используется только при необходимости.
---

## Итог

В рамках проекта:
- Построена базовая модель классификации.
- Реализован LLM-агент для принятия решений.
- Разработан многошаговый агент с маршрутизацией и веб-поиском.
